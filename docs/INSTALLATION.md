
# Installation Guide

## Requirements

- Node.js 20+
- PostgreSQL 16+
- Redis
- Google Chrome
- Git

---

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CyberPunk-Detection-Tool.git
```

---

## Backend

```bash
cd backend

npm install

npm start
```

---

## Database

Create PostgreSQL database

```
cyberpunk
```

Configure

```
.env
```

```
DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASSWORD=
```

redis-server
