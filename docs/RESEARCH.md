# Research

CyberPunk Detection Tool was developed as a research project in browser security and phishing detection.

Research Areas

- URL Analysis
- Phishing Detection
- Browser Fingerprinting
- Security Headers
- Threat Intelligence
- Cookie Security
- AI Explainability
- Trust Score Calculation

Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- False Positive Rate
- False Negative Rate
- Detection Latency

Future research includes:

- Large Language Models (LLMs)
- Retrieval-Augmented Generation (RAG)
- Graph Neural Networks
- Behavioral Threat Analysis
- Real-time Threat Correlation
- Explainable AI (XAI)
